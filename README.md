# Linear Regression - Advertising Dataset

A simple Linear Regression model trained on the Advertising dataset to predict Sales based on TV, Radio, and Newspaper ad spend.

## Dataset
- Source: [Kaggle - Advertising Dataset](https://www.kaggle.com/datasets/tawfikelmetwally/advertising-dataset)
- Features: TV, Radio, Newspaper
- Target: Sales

## What the Model Does
- Trains a Linear Regression model
- Outputs Coefficients, Intercept, and R² Score
- Predicts sales for custom input values
- Visualizes Actual vs Predicted Sales with Scatter Plot + Best Fit Line

## Libraries Used
- pandas
- scikit-learn
- matplotlib

## How to Run
1. Download the dataset from Kaggle
2. Open the notebook in Google Colab
3. Run all cells
